# Funbox

# Install instruction

## Clone the repo

```sh
git clone https://github.com/manuelep/funbox.git
```

## Intall package

```sh
python3 -m pip install -U funbox
```

## Run a game

```sh
python3 -m funbox.pokemon1.play
```
